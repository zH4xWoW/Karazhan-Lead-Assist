# Karazhan Lead Assist

***Updated for TBC Classic Anniversary!***

A World of Warcraft TBC Classic Anniversary addon designed to help raid leaders organize and manage Karazhan runs more efficiently. Karazhan Lead Assist provides quick access to raid-leading tools, making it easier to coordinate your group, manage assignments, and keep your raid organized from start to finish.

Built for players who regularly lead Karazhan groups, this addon is intended to reduce raid setup friction and give leaders a smoother way to handle common organizational tasks inside the instance.

## Features
- Raid leader focused Karazhan utility tools
- Easy-to-open in-game interface for managing raid information
- Streamlined organization for common Karazhan leadership tasks
- Support for raid setup and coordination during Karazhan runs
- Simple slash command access
- Lightweight design focused on practical usability
- Built specifically for TBC Classic Anniversary Karazhan content

## Installation
1. Download the addon
2. Extract the folder to your `World of Warcraft\_classic_era_\Interface\AddOns` or TBC Anniversary AddOns folder
3. Ensure the folder is named `KarazhanLeadAssist`
4. Launch the game and make sure the addon is enabled at the character selection screen

## Dependencies
None!

## Slash Commands
Use `/kla` to open or interact with the addon.

Example:
- `/kla` - Open the Karazhan Lead Assist interface

## Configuration
The addon is designed to be simple and quick to use in-game.

Configuration and usage are handled through:
1. The main addon window
2. Slash commands

## Notes for Developers

The addon uses the following structure:
- `KarazhanLeadAssist.toc` - Table of contents file with metadata
- `KarazhanLeadAssist.lua` - Main addon logic and UI functionality

Karazhan Lead Assist is intended to serve as a focused utility addon for raid leaders, with features centered around improving organization, consistency, and convenience during Karazhan runs.
