Karazhan Lead Assist
====================

Purpose
-------
A Karazhan-specific raid-leader helper for WoW TBC Classic Anniversary.
It is designed to complement boss mods like BigWigs by focusing on:
- Encounter explanations
- Assignment planning
- Trash reminders
- Announcement buttons
- Pull countdowns and ready checks

Install
-------
1. Copy the KarazhanLeadAssist folder into:
   World of Warcraft\_anniversary_\Interface\AddOns2. Restart the game or reload your UI.
3. Enable the addon on the character select AddOns button if needed.

Commands
--------
/kla                Open or close the window
/kla bosses         Jump to the boss notes list
/kla trash          Jump to the trash notes list
/kla utilities      Jump to utilities / prep tools
/kla plan           Announce the built-in plan and custom note
/kla assignments    Announce your saved assignment fields
/kla pull           Start a 10 second pull countdown
/kla target         Match your current target to a boss entry
/kla ready          Start a ready check

Workflow suggestion
-------------------
1. Open /kla before the raid starts.
2. Fill in the three assignment boxes for the current boss.
3. Add any team-specific note in the custom lead note box.
4. Click Announce Plan and Announce Assignments before pull.
5. Use Pull 10 for clean pulls.

Notes
-----
- The addon uses ## Interface: 20505.
- If Blizzard updates the TBC Anniversary interface number later, you may need to update the TOC file.
